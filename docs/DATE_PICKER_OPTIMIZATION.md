# 时间选择器优化 & 列表快速筛选功能

## 📅 时间选择器优化（v1.2.0）

### 新增快捷日期按钮

为了让日期筛选更加友好和高效，我们在日期选择器中添加了6个常用的快捷按钮：

#### 快捷按钮列表

| 按钮 | 功能 | 日期范围 |
|------|------|----------|
| 📅 今天 | 查看今天的日志 | 今天 0:00 - 23:59 |
| 📅 昨天 | 查看昨天的日志 | 昨天 0:00 - 23:59 |
| 📅 最近7天 | 查看最近一周 | 6天前 - 今天 |
| 📅 最近30天 | 查看最近一月 | 29天前 - 今天 |
| 📅 本月 | 查看本月至今 | 本月1日 - 今天 |
| 📅 上月 | 查看上个月 | 上月1日 - 上月最后一天 |

### 使用方法

#### 1. 快捷选择
点击任意快捷按钮，自动设置日期范围：
```
点击 "📅 今天" → 自动筛选今天的日志
点击 "📅 最近7天" → 自动筛选最近7天的日志
```

#### 2. 手动选择
如果需要特定的日期范围：
```
1. 在"开始日期"选择器中选择开始日期
2. 在"结束日期"选择器中选择结束日期
3. 自动应用筛选
```

#### 3. 清除筛选
点击 "✖ 清除" 按钮，清除所有日期筛选

## 🖱️ 列表快速筛选功能

### 功能说明

在日志列表（表格）中，以下列可以直接点击进行筛选，无需打开详情页：

### 可点击的列

#### 1. 域名列
- **点击效果**：筛选该域名的所有日志
- **视觉提示**：
  - 悬停时背景变浅紫色
  - 文字变紫色并加粗
  - 显示 🔍 图标
  - 轻微放大效果

#### 2. 方法列
- **点击效果**：筛选该请求方法（GET、POST等）
- **视觉提示**：
  - 悬停时放大 1.08 倍
  - 显示阴影效果
  - 显示 🔍 图标

#### 3. 状态码列
- **点击效果**：筛选该状态码范围（2xx、3xx、4xx、5xx）
- **视觉提示**：
  - 悬停时放大 1.08 倍
  - 显示阴影效果
  - 显示 🔍 图标

#### 4. 客户端 IP 列
- **点击效果**：筛选该 IP 的所有日志
- **视觉提示**：
  - 悬停时背景变浅蓝色
  - 文字变蓝色并加粗
  - 显示 🔍 图标
  - 轻微放大效果

### 交互流程

```
用户浏览日志列表
    ↓
鼠标悬停在可点击列（域名/方法/状态码/IP）
    ↓
看到视觉提示：
  - 背景颜色变化
  - 文字加粗/变色
  - 显示 🔍 图标
  - 轻微放大
    ↓
点击单元格
    ↓
自动应用筛选：
  - 更新筛选器
  - 刷新日志列表
  - 重置到第 1 页
    ↓
查看筛选结果
```

- 'last30days' - 最近30天
