# 快速筛选功能

## 功能说明

在日志列表中，直接点击域名、IP、方法、状态码等字段即可快速设置为筛选条件，无需打开详情页。

## 📋 可点击的字段

### 在日志列表中（主要功能）

表格中以下字段支持直接点击快速筛选：

### 1. 🌍 域名（Domain）
- **显示位置**：日志列表 → 域名列
- **点击效果**：自动筛选该域名的所有请求
- **示例**：点击 `fn.lztyun.cn` 后，只显示该域名的日志

### 2. 🔧 请求方法（Method）
- **显示位置**：日志列表 → 方法列
- **点击效果**：自动筛选该方法的所有请求
- **支持方法**：GET、POST、PUT、DELETE、PATCH、HEAD、OPTIONS 等
- **示例**：点击 `GET` 后，只显示 GET 请求

### 3. 📊 状态码（Status Code）
- **显示位置**：日志列表 → 状态码列
- **点击效果**：自动筛选该状态码范围的所有请求
- **筛选范围**：
  - 200-299 → 筛选 2xx 成功
  - 300-399 → 筛选 3xx 重定向
  - 400-499 → 筛选 4xx 客户端错误
  - 500+ → 筛选 5xx 服务器错误
- **示例**：点击 `404` 后，筛选所有 4xx 客户端错误

### 4. 🖥️ 客户端 IP（Client IP）
- **显示位置**：日志列表 → 客户端 IP 列
- **点击效果**：自动筛选该 IP 的所有请求
- **示例**：点击 `125.121.52.66` 后，只显示该 IP 的请求

### 在详情页中（辅助功能）

详情页中同样支持点击以下字段快速筛选，点击后会自动关闭详情页并应用筛选：

- 🌍 域名
- 🔧 请求方法
- 📊 状态码
- 🖥️ 客户端 IP

## 🎨 视觉提示

### 表格中的悬停效果
- 可点击的单元格（域名、方法、状态码、IP）：
  - 鼠标悬停时背景变浅蓝色
  - 单元格轻微放大（1.02倍）
  - 鼠标变为手型指针
  - 有标题提示（如"点击筛选此域名"）

### 不同字段的特殊效果
- **域名**：悬停时文字颜色加深，字重变粗
- **IP地址**：悬停时变为蓝色，字重变粗
- **方法徽章**：悬停时放大（1.05倍）并显示阴影
- **状态码徽章**：悬停时放大（1.05倍）并显示阴影

### 点击反馈
- 点击时有按下效果（轻微缩小）
- 立即应用筛选条件
- 重置到第 1 页
- 筛选器下拉框自动更新

### 详情页中的提示
- 可点击的字段会显示 🔍 图标
- 鼠标悬停时：
  - 背景变浅蓝色
  - 轻微向右移动
  - 显示提示文字

## 🎯 使用场景

### 场景 1：快速分析特定域名
1. 浏览日志列表
2. 看到感兴趣的域名
3. **直接点击域名** → 立即筛选该域名
4. 分析该域名的所有请求

### 场景 2：追踪可疑 IP
1. 发现一个异常请求
2. **直接点击客户端 IP** → 立即筛选该 IP
3. 查看该 IP 的所有历史请求
4. 分析该 IP 的行为模式

### 场景 3：分析特定请求方法
1. 浏览日志列表
2. **点击任意 POST 方法** → 筛选所有 POST 请求
3. 分析 API 调用情况

### 场景 4：错误分析
1. 看到一个 404 状态码
2. **直接点击状态码** → 筛选所有 4xx 错误
3. 快速定位所有问题页面

### 场景 5：深度分析（结合详情页）
1. 在列表中点击域名筛选
2. 点击某条记录的"查看详情"按钮
3. 在详情页中点击 IP 进一步筛选
4. 得到该域名+该IP的所有请求

## 💡 技巧与最佳实践

### 快速过滤工作流（推荐）
```
1. 浏览日志列表
2. 看到感兴趣的字段 → 直接点击
3. 立即查看筛选结果
4. 需要进一步细化？点击其他字段继续筛选
5. 需要查看详情？点击 🔍 按钮
```

### 组合筛选技巧
```
1. 点击域名 → 筛选特定域名
2. 点击方法 → 进一步筛选该域名的特定方法
3. 手动调整日期范围 → 精确定位时间段
4. 得到非常精确的结果
```

### 快速浏览模式
```
1. 点击一个域名 → 查看该域名流量
2. 选择"所有域名"清除筛选
3. 点击另一个域名 → 对比不同域名
4. 快速了解整体情况
```

### 深度分析模式
```
1. 在列表中点击域名初步筛选
2. 点击某条记录的详情按钮
3. 在详情页查看完整信息
4. 点击详情页的其他字段继续筛选
5. 层层递进分析问题
```

### 清除筛选
- 在筛选器下拉框中选择"所有XXX"即可清除该筛选条件
- 例如：选择"🌍 所有域名"清除域名筛选

## 🔧 技术实现

### 前端实现
- **组件**：`LogTable.js`
- **点击处理函数**：
  - `handleDomainClick(domain)` - 处理域名点击
  - `handleIpClick(ip)` - 处理 IP 点击
  - `handleMethodClick(method)` - 处理方法点击
  - `handleStatusClick(status)` - 处理状态码点击

### 样式实现
- **表格样式**：`.clickable-cell` - 应用于表格单元格
- **详情页样式**：`.clickable-filter` - 应用于详情页字段
- **交互效果**：
  - `:hover` - 悬停效果
  - `:active` - 点击效果
  - `transform` - 位移和缩放动画
  - `transition` - 平滑过渡

### 工作流程（表格点击）
1. 用户点击表格中的可点击单元格
2. 触发对应的处理函数
3. 更新筛选状态（`setXxxFilter`）
4. 重置到第 1 页（`setCurrentPage(1)`）
5. 自动刷新日志列表

### 工作流程（详情页点击）
1. 用户在详情页点击可点击字段
2. 触发对应的处理函数
3. 更新筛选状态
4. 关闭详情页（`handleCloseModal()`）
5. 重置到第 1 页
6. 自动应用筛选并显示结果

## 🔄 与现有功能的集成

### 与筛选器集成
- 点击快速筛选后，对应的筛选器下拉框会自动更新
- 可以继续手动调整其他筛选条件
- 支持多维度组合筛选

### 与分页集成
- 应用筛选后自动重置到第 1 页
- 支持优化模式和全量模式
- 优化模式会触发后端重新加载数据

### 与搜索集成
- 可以先搜索，再点击详情快速筛选
- 筛选条件会叠加
- 清除某个筛选不影响其他条件
